# 🧮 Продвинутый Калькулятор

Красивый и функциональный калькулятор на Python с пошаговым решением математических выражений.

## ✨ Возможности

### 🎯 Основные функции:
- **Базовые операции**: сложение, вычитание, умножение, деление
- **Проценты** и **изменение знака**
- **Работа с десятичными дробями**

### 🔬 Научные функции:
- **Тригонометрия**: sin, cos, tan
- **Логарифмы**: log (десятичный), ln (натуральный)
- **Степени**: √ (квадратный корень), x², xʸ
- **Константы**: π (пи), e (число Эйлера)
- **Скобки** для сложных выражений

### 🎨 Дизайн:
- **Две темы**: светлая и темная
- **Красивый градиентный интерфейс**
- **Анимированные кнопки**
- **Современный дизайн**

### 📋 Пошаговое решение:
- **Детальное описание** каждого шага вычисления
- **Анализ выражения** и проверка скобок
- **Пошаговое решение** сложных формул
- **Включение/выключение** пошагового режима

### 📚 Дополнительные функции:
- **История вычислений** (последние 10 операций)
- **Большой дисплей** для удобного ввода
- **Профессиональная типографика**

## 🚀 Установка

1. **Клонируйте репозиторий**:
```bash
git clone <repository-url>
cd calculatorpro
```

2. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

3. **Запустите калькулятор**:
```bash
python calculator.py
```

## 📖 Использование

### Базовые операции:
- Вводите цифры с кнопок 0-9
- Используйте `+`, `-`, `×`, `÷` для операций
- Нажмите `=` для получения результата
- Используйте `C` для очистки

### Научные функции:
- **Тригонометрия**: `sin(45)`, `cos(30)`, `tan(60)`
- **Логарифмы**: `log(100)`, `ln(2.718)`
- **Степени**: `√16`, `5²`, `2³` (через `2**3`)
- **Константы**: `π`, `e`

### Сложные выражения:
```
(2 + 3) × 4 - 5 ÷ 2
sin(30) + cos(60) × √4
log(100) + ln(e²)
```

### Пошаговое решение:
1. Введите выражение
2. Убедитесь, что галочка "Пошагово" включена
3. Нажмите `=`
4. Справа появится детальное решение

## 🎮 Управление

### Кнопки:
- **C** - Очистить дисплей
- **±** - Изменить знак числа
- **%** - Процент от числа
- **()** - Скобки для группировки операций
- **.** - Десятичная точка

### Научные кнопки:
- **sin, cos, tan** - Тригонометрические функции
- **log, ln** - Логарифмы
- **√** - Квадратный корень
- **x²** - Возведение в квадрат
- **xʸ** - Возведение в степень
- **π, e** - Математические константы

### Тема:
- Выберите "Светлая" или "Темная" в выпадающем меню
- Интерфейс автоматически применит новую тему

## 🛠 Технические детали

- **Язык**: Python 3.8+
- **Фреймворк**: PyQt6
- **Архитектура**: Объектно-ориентированная
- **Парсер**: Собственный алгоритм пошагового решения

## 📝 Примеры использования

### Пример 1: Базовые операции
```
Ввод: 15 + 25 × 2
Пошагово:
📝 Анализ выражения: 15 + 25 × 2
🔢 Умножение: 25 × 2 = 50
🔢 Сложение: 15 + 50 = 65
✅ Финальный результат: 65
```

### Пример 2: Скобки и тригонометрия
```
Ввод: (30 + 45) × sin(60)
Пошагово:
📝 Анализ выражения: (30 + 45) × sin(60)
📦 Решаем выражение в скобках: 30 + 45
🔢 Сложение: 30 + 45 = 75
📦 Результат в скобках: 75
🔄 Заменяем скобки: 75 × sin(60)
✅ Финальный результат: 64.95
```

## 🐛 Устранение неполадок

**Проблема**: Калькулятор не запускается
**Решение**: Убедитесь, что установлен PyQt6: `pip install PyQt6`

**Проблема**: Неверный результат вычисления
**Решение**: Проверьте правильность расстановки скобок и используйте пошаговый режим для анализа

**Проблема**: Научные функции не работают
**Решение**: Убедитесь, что используете правильный синтаксис: `sin(30)`, а не `sin30`
